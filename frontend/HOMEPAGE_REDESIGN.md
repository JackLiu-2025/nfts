  - 分类名称（大标题）

  - 导航箭头
- **渐变配色**：
  - 艺术：紫色→粉色→紫色
  - 收藏品：蓝色→青色→蓝色
  - 游戏：黄色→橙色→红色
  - 音乐：绿色→翠绿→青绿

## 🎭 动画效果

### 进入动画
- Hero区域：从下往上淡入(0.8s)
- 徽章：缩放淡入(延迟0.2s)
- 标题：分层淡入(延迟0.3s, 0.4s)
- 描述：淡入(延迟0.5s)
- 按钮：从下往上淡入(延迟0.6s)

### 滚动动画
- 所有section使用`whileInView`
- 只触发一次(`viewport={{ once: true }}`)
- 从下往上淡入效果

### 悬停动画
- 分类卡片：上浮、缩放、旋转、光晕
- 按钮：箭头向右移动
- 平滑过渡(duration-300/500)

## 🌈 视觉层次

### 字体大小
- 主标题：6xl/8xl (96px/128px)
- Section标题：4xl/5xl (48px/64px)
- 分类标题：2xl (24px)
- 描述文字：lg (18px)

### 间距
- Section间距：mb-32 (128px)
- 标题间距：mb-16 (64px)
- 卡片间距：gap-6 (24px)

### 颜色使用
- 主色：cyber-cyan (#00f5ff)
- 渐变：text-gradient
- 文字：white/60 (60%透明度)
- 边框：white/10, white/30

## 📱 响应式设计

- 移动端：1列分类卡片
- 平板：2列分类卡片
- 桌面：4列分类卡片
- 标题在移动端自动缩小

## 🎯 用户体验

1. **视觉引导** - 从上到下的自然阅读流
2. **交互反馈** - 所有可点击元素都有明显的hover效果
3. **性能优化** - 使用`viewport={{ once: true }}`避免重复动画
4. **可访问性** - 保持良好的对比度和可读性

## 🚀 技术实现

- Framer Motion - 所有动画
- Tailwind CSS - 样式和渐变
- Lucide React - 图标
- React Router - 导航

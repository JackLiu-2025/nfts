# 🚀 自动跳转快速参考

## 操作跳转一览表

| 操作 | ⏱️ 倒计时 | 📍 跳转目标 | 💬 提示文字 |
|------|----------|-----------|-----------|
| 🎨 铸造 NFT | - | 停留当前页 | - |
| 🏷️ 挂单出售 | 3秒 | 个人中心-出售中 | "Redirecting to your listings in 3s..." |
| ❌ 取消挂单 | 3秒 | 市场页面 | "Redirecting to marketplace in 3s..." |
| 🛒 购买 NFT | 2秒 | 个人中心-收藏 | "Redirecting to your collection in 2s..." |
| 🔥 销毁 NFT | 立即 | 个人中心 | - |

## URL 路径

```
挂单出售 → /profile?tab=listed
取消挂单 → /marketplace
购买 NFT → /profile?tab=collected
销毁 NFT → /profile
```

## 个人中心标签

- `?tab=created` - 我的创作
- `?tab=collected` - 我的收藏
- `?tab=listed` - 出售中

## 设计理念

### 🏷️ 挂单 → 出售中
✅ 立即查看挂单状态  
✅ 确认价格信息  
✅ 管理所有挂单

### 🛒 购买 → 收藏
✅ 立即查看新收藏  
✅ 快速反馈  
✅ 提升满意度

### ❌ 取消挂单 → 市场
✅ NFT 已下架  
✅ 继续浏览  
✅ 发现新 NFT
